# course-post

### 一个python爬虫，用来获取课表数据和解析数据

### 配置项目环境

#### 启动虚拟环境

```powershell
python -m venv .venv
```

</a>

#### 激活虚拟环境

```powershell
windows
.\.venv\Scripts\Activate.ps1
Macos
source .venv/bin/activate
```

#### 安装依赖

```shell
pip install -r requirements.txt
```

#### 启动本地爬取课表

```shell
python main.py -local
```

#### 启动接口服务

```shell
python server.py
```
